# 子代理任務書：war-vetustapolis-memory

## 目標
為 **維特魯斯（Vetustapolis）** 補充重大歷史戰爭的定居點記憶。維特魯斯是安納克大陸最重要的城市之一，作為多場關鍵戰爭的直接戰場或核心受害者，其 `settlement.yaml` 中的事件覆蓋嚴重不足。

## 輸出檔案
`data/01-west-seraphion/locations/vetustapolis/settlement.yaml`

## 核心設定事實（來自 source）

### 維特魯斯的戰爭史
1. **第二次白色洪流（第451年）**：霜生攻陷維特魯斯，將宏偉的神殿凍結成冰雕，釘在太陽圓盤上的祭司成為永恆的警告。數十萬難民南逃。
2. **鐵冠起義（第726年）**：巴爾塔薩的赤土農民軍攻陷維特魯斯，艾森瓦爾五世自焚於宮殿。這是雙帝分立的直接原因。
3. **大洪水（第0年）**：雖非戰爭，但大洪水塑造了城市的地基與排水系統，也奠定了宗教起源。

### 當前（第825年）的維特魯斯戰爭記憶
- 城市中的「六聖徒神殿」是極北會戰的紀念，但 source 描述為教會宣傳的產物
- 「劍誓池」是東遷的起點，馬庫斯親王在此祭拜艾森瓦爾五世骨灰
- 皇宮地下可能仍有艾森瓦爾五世自焚的焦痕
- 被凍結的神殿遺跡可能仍存在（或未完全融化）

## 執行規則
- 每個事件必須有獨立的 `id`、年份、`name`、`description`
- 事件應體現「定居點層級的戰爭記憶」——不是戰爭本身的宏大敘事，而是**城市空間如何被戰爭改變**
  - 建築遺跡（凍結的神殿、焚毀的宮殿、加固的城牆）
  - 節日與儀式（紀念日、遊行、禁忌）
  - 人口結構（難民後裔、傭兵後代、霜生恐懼症）
  - 經濟影響（重建稅、戰爭債券、防禦工事開支）
- 所有推斷內容標註 `# inferred: true`
- 不得與現有 `events` 重複
